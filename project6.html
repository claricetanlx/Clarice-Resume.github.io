Project Overview

A 3-month academic project where we designed and developed a fully functional website for Summer Paws, a fictional pet care service. Working with a partner, we created an interactive, responsive website featuring service pages, booking forms, and animated UI elements using front-end technologies. The project combined design principles with practical coding implementation.

Key Features

Interactive UI

Implemented JavaScript for dynamic elements like image sliders, form validation, and mobile menus

Theme Consistency

Maintained cohesive branding through custom color variables and typography


Challenges & Solutions

Creating responsive layouts that worked seamlessly across devices proved challenging, especially with complex grid structures. We overcame this by adopting a mobile-first CSS approach, using Flexbox for flexible components and carefully tested media queries for each breakpoint. The booking form's JavaScript validation initially failed in older browsers until we implemented a hybrid solution combining HTML5 validation with custom JavaScript checks for better compatibility.
